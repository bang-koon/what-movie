### 주요 기능
- 일일 박스오피스 순위 표시
- 영화 상세 정보 제공 (감독, 배우, 줄거리 등)
- 왓챠, 로튼토마토 평점 정보
- 반응형 UI
- ISR
- DB 데이터 캐싱
  
### 스택
- Next.js 13
- TypeScript
- MongoDB
- SCSS

### API
- 영화진흥위원회 (KOBIS)
- 한국영화데이터베이스 (KMDB)
- 왓챠/로튼토마토 (크롤링)
